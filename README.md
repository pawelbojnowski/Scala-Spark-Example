# ScalaSparkExample

### RUN

This application need connections to Cassandra, MariaDB and CockroachDB.

In directory ``dbs`` you can find script ``create_dbs.sh`` which creates docker containers with tables and some examples data
for both DBs;)

- SBT
- Scala
